# tradedata-Engine
O TradeData-Engine é um pipeline de dados ponta-a-ponta (ETL) projetado para coletar, tratar e servir dados do mercado financeiro e criptoativos. O foco do projeto é aplicar o conceito de Modern Data Stack (MDS) para garantir que traders e sistemas automatizados consumam dados confiáveis, normalizados e de baixa latência.
